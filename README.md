# nintex-code-test
code test for nintex


Hi There, 
use instructions:

1- clone the repository:
	>git clone git@github.com:desphilboy/nintex-code-test.git

2- install needed packages:
	> cd nintex-code-test
	> npm install

3- test the project:
	> npm run test

4- run the project:
	> npm start

5- see the web page:
	type this url in google chrome: http://localhost:9080/
